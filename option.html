安全保护

  • cloud 云信誉评估 实时拦截危险网站，保护您不受钓鱼、恶意网址的威胁
  • file_download 下载保护 自动阻止从危险网站上的下载
  • pan_tool 隐私保护 阻止分析网站对您的网络活动进行跟踪或记录
  • warning 潜在威胁 开启后将阻止不受欢迎的网站、信誉评估较低的网站
  • monetization_on 挖矿脚本拦截 帮助您避免电脑被恶意网站用于挖矿（比特币）

实验室

  • search 书签搜索 您在搜索引擎搜索时自动帮助您检索书签
  • announcement 安全提示 在访问银行、电商等容易被冒充的网站时弹出安全提示

  • bug_report 数据收集与使用 开启后会在输入密码时自动弹出虚拟键盘，保护您的输入不被监听

  • memory 内存优化 自动对长时间不访问的标签页进行内存优化
